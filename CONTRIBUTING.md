# Contributing

Thanks for your interest in EMMA.

## Scope of this public preview
- Documentation fixes and packaging polish are welcome.
- Interface-level improvements and clearer examples are welcome.
- Core implementation remains withheld until acceptance.

## Before opening a PR
- Keep secrets, local paths, private seeds, and benchmark-private artifacts out of the tree.
- Keep changes consistent with the public preview policy in `README.md`.
- Prefer small, reviewable patches with a clear purpose.

## Suggested workflow
1. Open an issue describing the change.
2. Keep the patch focused on one surface area.
3. Update docs when user-facing behavior changes.
4. Add or update lightweight checks when applicable.
